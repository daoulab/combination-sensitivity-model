# Code for Figures 2–8 and Supplementary Figures S1–S6

## What this code is for
This repository contains the MATLAB source code used to generate the main simulation figures (**Figures 2–8**) and supplementary figures (**Figures S1–S6**) for the study on stimulus-specific combination sensitivity. The code implements biophysically realistic conductance-based network models, reduced illustrative models, parameter sweeps, and plotting routines used to reproduce the figures reported in the paper.

## Which paper it belongs to
**Post-synaptic facilitation and network dynamics underlying stimulus-specific combination sensitivity**

Accepted in **iScience**.

## Authors
- **Zeina Merabi**
- **Arij Daou**

## What software is needed
- **MATLAB** is required to run the code.
- A relatively recent version of MATLAB is recommended.
  **MATLAB R2020b or newer** is preferred because some scripts use plotting functions such as `boxchart`, `swarmchart`, and `heatmap`.
- No external toolboxes or third-party packages are intended to be required beyond standard MATLAB functionality for:
  - ODE integration (`ode23`, `ode113`)
  - matrix handling
  - standard plotting

## Repository organization
Each figure is generated by a main MATLAB script. In the integrated versions prepared for this repository, helper functions and model equations have been folded into single self-contained files whenever possible.

## How to run the main scripts
1. Open MATLAB.
2. Change the current folder to the directory containing the figure scripts.
3. Run the desired figure script from the MATLAB command window.

Examples:
```matlab
Figure_2
Figure_3
Figure_S6
```

Some scripts return a `results` structure; others primarily generate the figure directly.

## What each main file does

### Main figures

#### `Figure2.m`
Generates the simple postsynaptic-facilitation illustration showing how stimulus order and timing determine successful or failed integration in a reduced conceptual network.

#### `Figure3.m`
Generates the figure showing combination-sensitive behavior across inter-stimulus delays, including representative network activity and the delay-versus-CSN spike count relationship.

#### `Figure4.m`
Generates the figure analyzing CSN intrinsic priming across delays, including the delay dependence of intrinsic currents such as `I_H` and `I_CaT` and their relationship to spike output.

#### `Figure5.m`
Generates the unsuccessful-integration network simulations:
- panel A: A alone
- panel B: the uploaded second condition as implemented in the source scripts
- panel C: reversed stimulus order

#### `Figure6.m`
Generates the figure showing CSN post-synaptic priming dynamics at a 50 ms inter-stimulus delay under different intrinsic conductance settings (`g_H` and `g_CaT`).

#### `Figure_7_integrated.m`
Generates the parameter-sweep and heatmap figures associated with intrinsic, synaptic, delay, duration, and intensity interactions. This integrated script embeds the matrix data directly in the code so it runs without separate `.txt` files.

#### `Figure8.m`
Generates the figure showing how stimulus duration and intensity shape CSN output:
- duration line plots
- duration heatmap
- intensity line plots
- intensity heatmap

### Supplementary figures

#### `Figure_S1.m`
Generates Supplementary Figure S1, containing boxplots and swarmplots for distributions of intrinsic and synaptic conductance parameters used in the model.

#### `Figure_S2.m`
Generates Supplementary Figure S2, a reduced three-neuron simple summation model used as an illustrative comparison to the full network.

#### `Figure_S3.m`
Generates Supplementary Figure S3, a reduced simple inhibitory model used as another illustrative comparison.

#### `Figure_S4.m`
Generates Supplementary Figure S4 with two panels:
- panel A: A+A with 10 ms delay
- panel B: B+B with 10 ms delay

#### `Figure_S5.m`
Generates Supplementary Figure S5, the inverse-order counterpart of the intrinsic-current interaction figure, showing CSN dynamics under reversed stimulus order.

#### `Figure_S6.m`
Generates Supplementary Figure S6, a long-delay A→B condition with elevated CSN T-type calcium conductance.
